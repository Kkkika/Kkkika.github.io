语法分解，代码片段解析

  • 这是一个相邻兄弟选择器（+），选中两个连续的 .cm-line 元素
  • 前半部分：.cm-line:not(:has(> br:only-child)) - 前一个行
  • 后半部分：.cm-line:has(> br:only-child) - 后一个行
代码片段 含义 作用
.cm-line 类选择器 选中所有Obsidian编辑器行
:not(...) 否定伪类 排除符合括号内条件的元素
:has(...) 关系伪类 必须包含符合括号内条件的子元素
> 子选择器 直接子元素（不是后代）
br:only-child br标签且是唯一子元素 该行只有一个<br>元素
+ 相邻兄弟选择器 选择紧跟在后的兄弟元素

Windows 任务计划程序（推荐）

  1. 打开"任务计划程序"
    • 按 Win + R，输入 taskschd.msc
    • 或搜索"任务计划程序"
  2. 创建基本任务：
    • 右侧点击"创建基本任务"
    • 名称：Hexo自动部署
    • 触发器：选择"每天"，设置时间13:00:00
    • 操作：选择"启动程序"
      • 程序或脚本：cmd.exe
      • 参数：/k "cd C:\Code\KikaBlog\blog && hexo clean && hexo g && hexo d"
      • 起始于：C:\Code\KikaBlog\blog
